## 프로젝트 개요

이 프로그램은 공항 방문객이 간단한 질문에 답하고 사진을 촬영하면,  
선택한 답변과 얼굴 이미지를 바탕으로 조선시대 인물 콘셉트의 결과물을 생성하는 체험형 콘텐츠입니다.

## 주요 기능

- 질문 기반 성향 선택
- 실시간 사진 촬영
- 얼굴 이미지 기반 AI 합성
- 조선시대 콘셉트 결과물 생성
- 전시 및 체험형 키오스크 환경 대응

## 화면 구성

| 화면 | 설명 |
|---|---|
| 메인 화면 | 프로그램 시작 및 안내 |
| 질문 선택 화면 | 사용자 응답 수집 |
| 사진 촬영 화면 | 얼굴 이미지 촬영 |
| 결과 화면 | 조선시대 인물 스타일 합성 결과 출력 |
